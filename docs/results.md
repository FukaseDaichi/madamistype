# 全体の傾向

今回の回答は、現時点で 206件 でした。
タイプ分布を見ると、特に多かったのは次の3タイプです。
• 扇動役者(TFEI) … 40件
• 大女優(TREI) … 28件
• 潜入記者(OFEI) … 26件

この3タイプだけで、全体のかなり大きな割合を占めていました。
その次に多かったのは、余白の魔術師(OREI)、尋問官(TFLI)、潜伏刑事(OFLI) あたりです。

全体としては、
• 人を動かしたり場を揺らしたりするのが好き
• 情報を拾って回るのが好き
• 物語や感情をしっかり味わいたい

といった方向のプレイスタイルの方に多くご協力いただいた印象でした。

全体でよく名前が挙がった作品

自由記述なので表記ゆれは多少ありますが、全体で特によく名前が挙がっていたのはこのあたりでした。1. この闇をあなたと 2. ふわふわクリームさらにジューシー 3. シュレディンガーの密室 4. 鬼哭館の殺人事件 5. エイダ

このほか、透きとおる青の証明、ウルイの血族、エンドロールは流れない あたりも複数のタイプから名前が出ていました。
全体として、特定タイプだけの作品というより、幅広い層から好かれている作品が見えてきた印象です。

自由記述で多かった意見

作品の話とは別に、自由記述ではかなり共通した意見も見られました。
いちばん多かったのは、設問の意味や意図が少し分かりづらい という声です。

具体的には、
• 「推理重視」は「推論重視」の方が伝わりやすいかも
• 何を想定した質問なのか掴みにくい設問がある
• 「おすすめのマダミス」が“自分のおすすめ”なのか“そのタイプ向けおすすめ”なのか分かりづらい
• 証拠品／証拠カード／情報カードなど、表記は統一した方がよさそう
• 引いた役やキャラクターによって回答がぶれやすい
• 憑依型RPの人には答えにくい質問もありそう

といった意見が目立ちました。

一方で、
• 「面白い診断だった」
• 「自分のタイプが分かって楽しかった」
• 「結果に納得感があった」
• 「話題作りにちょうどいい」

という好意的な感想もかなり多く、企画そのものはしっかり楽しんでもらえていたようです。

また、少数ですが、
• 「作品との相性より、どんな役が好きかとの相性の方が大きいかも」
• 「キャラクターへの共感を軸に遊んでいる」
• 「推理も物語もどちらも高濃度な作品が好き」

といった、今後の診断の広げ方につながりそうな意見もありました。

今回のアンケート結果をざっくりまとめると、次のような傾向が見えてきました。
• 多かったのは 扇動役者・大女優・潜入記者
• 全体で名前がよく挙がったのは
この闇をあなたと／ふわふわクリームさらにジューシー／シュレディンガーの密室／鬼哭館の殺人事件／エイダ

• 潜入記者 → この闇をあなたと / エイダ
• 余白の魔術師 → 透きとおる青の証明 / 英国探偵とウォルターの遺産

• 自由意見では 診断企画そのものは好評。ただし設問文の分かりやすさは改善余地あり

「自分はこういう遊び方が好きなのかも」と振り返るきっかけになるのが面白いところです。
